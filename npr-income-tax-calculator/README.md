# NPR Employment Income Tax Calculator

A static GitHub Pages calculator for Nepal employment income tax slabs in NPR.

## Features

- Slab-wise annual tax calculation
- Monthly tax and monthly after-tax income
- Optional 1% first-slab waiver
- Session-only storage with `sessionStorage`
- No backend, cookies, or shared database
